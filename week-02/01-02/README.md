## 十进制数与R进制数之间的转换

### R进制数 => 十进制数

* 按 ”权“ 展开

### 十进制数 => 二进制数、再将二进制转换为16或8进制

**整数部分和消暑部分分别转换**

* 整数： 除基取余，上右下左
* 小数： 乘基取整，上左下右

## 定点数和浮点数
